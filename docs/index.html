HomeKeeper

家庭の管理をもっとスマートに

プッシュ通知で家事のリマインダーやスケジュールを管理。
毎日の暮らしをサポートするiOSアプリです。

GitHub で見る

主な機能

🔔

プッシュ通知

タスクの期限や予定をプッシュ通知でお知らせ。忘れがちな家事も見逃しません。

📋

タスク管理

掃除・洗濯・買い物などをカテゴリ別に整理。家族で共有も可能です。

📅

スケジュール

繰り返しタスクの自動スケジュール設定。定期的な家事を楽に管理できます。

HomeKeeper について

HomeKeeperは、SwiftUI で構築されたiOS向け家庭管理アプリです。
シンプルで使いやすいインターフェースと、プッシュ通知による確実なリマインダー機能で、
日々の家事管理をサポートします。
