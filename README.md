# HomeKeeper

HomeKeeperは、家庭の管理をサポートするiOSアプリです。プッシュ通知を活用して、家事のリマインダーやスケジュール管理を行います。

## 機能

- 家事タスクの管理・リマインダー
- プッシュ通知によるスケジュール通知
- カテゴリ別タスク整理

## 技術スタック

- Swift / SwiftUI
- UserNotifications framework
- Swift Package Manager

## 開発環境

- Xcode 15+
- iOS 17+
- Swift 5.9+

## セットアップ

```bash
git clone https://github.com/ivytech-dev/HomeKeeper.git
cd HomeKeeper
open HomeKeeper.xcodeproj
```
